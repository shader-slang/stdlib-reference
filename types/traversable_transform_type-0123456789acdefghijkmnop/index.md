# enum TRAVERSABLE_TRANSFORM_TYPE

## Description

Type representing transform types in multi-level traversal.


## Values 

####  <a id="decl-TRAVERSABLE_TRANSFORM_TYPE_NONE"></a>_TRAVERSABLE_TRANSFORM_TYPE_NONE = 0_
No transformation applied

####  <a id="decl-TRAVERSABLE_TRANSFORM_TYPE_STATIC"></a>_TRAVERSABLE_TRANSFORM_TYPE_STATIC = 1_
Static matrix transformation

####  <a id="decl-TRAVERSABLE_TRANSFORM_TYPE_MATRIX_MOTION"></a>_TRAVERSABLE_TRANSFORM_TYPE_MATRIX_MOTION = 2_
Matrix motion transformation

####  <a id="decl-TRAVERSABLE_TRANSFORM_TYPE_SRT_MOTION"></a>_TRAVERSABLE_TRANSFORM_TYPE_SRT_MOTION = 3_
SRT (scale/rotation/translation) motion transformation

####  <a id="decl-TRAVERSABLE_TRANSFORM_TYPE_INSTANCE"></a>_TRAVERSABLE_TRANSFORM_TYPE_INSTANCE = 4_
Instance transformation
