# RWByteAddressBuffer\.Load4Aligned

## Description

Load four 32-bit unsigned integers from the buffer at the specified location with alignment
of <span class='code'>uint4</span>, which is 16.

## Parameters

####  <a id="decl-location"></a>location  : uint

## Return value
